README

- To check if you can run a hotspot on you PC open "cmd" from Start menu and type

	" netsh wlan show drivers "
  just read if hosted network is supported or not

- After you started the hotspor, go to Network Settings, select the Ethernet connection and click on "Properties"
- In the "share" panel, you must check and share your new connection
- Now you can use this software to create a Network ^^ 
(Remember to start it as Administrator)
